Privacy Policy

Last Updated: January 1, 2024
App Version: 1.0.0
App Name: Docifire Document Scanner

Quick Summary

Docifire respects your privacy! All file processing happens locally on your device.

We do NOT store, upload, or share your files with external servers, except for:

  • Background Removal AI: Uses Remove.bg API (files are deleted after processing)
  • Ads: Google AdMob (anonymous data only)

1. Information We Collect

We collect minimal information to provide our services:

  • Files for Processing: Only during active use, never stored
  • App Usage Data: Anonymous statistics to improve features
  • Device Information: Basic info for compatibility
  • Crash Reports: Error logs to fix bugs
  • Ad Interaction Data: Through Google AdMob (anonymous)

2. How We Use Your Information

Your data is used only for:

  1. Local File Processing: All conversions happen on your device
  2. Background Removal: Images sent temporarily to Remove.bg API
  3. App Improvement: Anonymous usage analytics
  4. Bug Fixing: Crash reports to improve stability
  5. Ad Personalization: Through Google AdMob (you can opt-out)

3. App Features & Data Handling

PDF Operations

All offline: Convert, merge, split, compress, lock/unlock

ZIP Operations

All offline: Create, extract, protect, view, repair

Image Processing

Mostly offline: Format conversion, OCR, scanner

Online: Background removal via Remove.bg API

4. Third-Party Services

External Services We Use

  • Remove.bg API: For AI background removal only
  • Google AdMob: For displaying ads (configurable)
  • Firebase Remote Config: For managing API keys securely

Important: Files sent to Remove.bg are deleted immediately after processing.

5. Data Protection & Security

Our Security Measures

  • Local Processing: 90% of features work completely offline
  • No File Storage: We don't keep copies of your files
  • Encrypted Communication: HTTPS for all API calls
  • API Key Protection: Keys stored securely in Firebase
  • Minimal Permissions: Only what's necessary for features

6. Your Rights & Choices

You have the right to:

  • Opt-out of Ads: Through Google AdMob settings
  • Disable Online Features: Use only offline features
  • Revoke Permissions: At any time in app settings
  • Clear App Data: Delete all locally stored data
  • Request Information: About what data we have

7. Children's Privacy

Our app is suitable for users of all ages. We do not knowingly collect personal information from children under 13.

8. Policy Updates

We may update this policy. Changes will be posted here with updated "Last Updated" date.
